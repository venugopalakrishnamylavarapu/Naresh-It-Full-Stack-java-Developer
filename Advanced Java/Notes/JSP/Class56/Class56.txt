Converted Servlet (or) JSP "Servlet" Life Cycle Methods:
=======================================================
follows the same order

1)jspInit()	->for first request

2)_jspService()	->repeats for every request

3)jspDestroy()	->When application deployed or server destroyed.

jspInit() method & jspDestroy() method are the part of jakarta.servlet.jsp.JspPage interface.

_jspService() method is a part of jakarta.servlet.jsp.HttpJspPage interface.
