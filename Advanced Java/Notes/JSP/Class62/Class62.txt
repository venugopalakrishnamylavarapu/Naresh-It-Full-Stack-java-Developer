9)<jsp:param>:
==============
it is used to specify the parameter with name and value.
Syntax:
------
	<jsp:param ......>
	</jsp:param>
		(or)
	<jsp:param ....../>
Here ...... is Attributes.

Attributes:
==========
1)name="........" --->..... is name of the parameter.
2)value="........" --->..... is value of the parameter.

Example:
=======
<jsp:param name="message" value="Welcome"/>
This parameter can be retrieved in applet by using the following method of java.applet.Applet class.

public String getParameter(String);

This action must be used in <jsp:params> action only.

Example:
=======
<jsp:param type="applet" code="demo.MyApplet" width=""400" height="400">
<jsp:params>
<jsp:param name="message" value="Welcome"/>
</jsp:params>
<jsp:fallback>
Unable to load an applet
</jsp:fallback>
</jsp:plugin>

JSP EL:
=======
JSP EL stands for Jakarta Server Pages Expression Language.

JSP EL introduced with JSTL in JSP 2.0 version to simplify the JSP.

JSTL stands for JSP Standard Tag Library.

JSP EL can be used independently and it can also be used with JSTL.

By using JSP EL & JSTL in a JSP,JSP becomes 100% tag based application.

JSP EL is an alternative to Expressions.
JSTL is an alternative to Declarations & Scriplets.

The pattern that identifies JSP Expression Language is ${..........}.

By default JSP Expression Language enabled in JSP.

To disable,JSP Expression Language,use the following:
<%@page isELIgnored="true"%>
