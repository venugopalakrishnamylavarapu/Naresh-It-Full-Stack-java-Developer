Servlet Listners:
================
Servlet listners are interfaces that contain event handlers(methods) to handle events(classes & objects) which
are generated by web container.

List of ServletListeners:
=========================
1)ServletContextListener
2)ServletContextAttributeListener
3)HttpSessionListener
4)HttpSessionAttributeListener ..etc

ServletContextListner:
----------------------
it contains the following event handlers to handle ServletContextEvent,which is generated by
web container.

1)public default void contextInitialized(ServletContextEvent);
2)public default void contextDestroyed(ServletContextEvent);


ServletContextEvent is generated by web container whenever ServletContext is created & destroyed.

ServletContext is created by web container whenever web application is deployed on server.

ServletContext is destroyed by web container whenever web application is undeployed from server.

The following series of actions done by web container whenever web application is deployed on
server:
1)ServletContext is created.
2)ServletContextEvent is generated.
3)ServletContextEvent is passed to contextInitialized() method.
4)contextInitialized() method body executed.
*Note:contextInitialized() method is present in ServletContextListener interface.

The following series of actions done by web container whenever web application is undeployed from 
server:
1)ServletContext is destroyed.
2)ServletContextEvent is generated.
3)ServletContextEvent is passed to contextDestroyed() method.
4)contextDestroyed() method body executed.
*Note:contextDestroyed() method is present in ServletContextListener interface.
