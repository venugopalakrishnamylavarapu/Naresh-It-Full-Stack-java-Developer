-------------------------------------------------------------------------------------------

There are two types of concurrency control on ResultSet
1)public static final int CONCUR_READ_ONLY;
2)public static final int CONCUR_UPDATABLE;

These all static members in java.sql.ResultSet interface.

CONCUR_READ_ONLY suppourts only read operation concurrently where as CONCUR_UPDATABLE suppourts 
all operations concurrently.
