jakarta.servlet.ServletResponse

Methods:
 public abstract ServletOutputStream getOutputStream() throws IOException;
-> here ServletOutputStream is Byte stream(audio,vedio etc..)

 public abstract PrintWriter getWriter() throws IOException;
-> here PrintWriter is Character Stream(only text)

=>The above methods are used to send the response to a web client(web browser)



program of servlet(once understand this program model every thing will be same)
=================
FirstServlet.java
-----------------

import java.io.*;
import jakarta.servlet.*;

class FirstServlet extends GenericServlet
{
public void service(ServletRequest req,ServletResponse res)
	{
		try{
		PrintWriter pw=res.getWriter();
		pw.println("Welcome to Servlets");
		}catch(Exception e)
		{	
			e.printStackTrace();	
		}
	}
}


after running this u need to create xml
for that go to cmd and type(with compiling of the program too)

cd ..(it goes to WEB-INF)

Steps to start tomcat Web Server:
=================================
1)Open tomcat Folder
Example:C:\Program Files\Apache Software Foundation\Tomcat 10.0
2)Open bin folder in tomcat folder
3)Double click on tomcat icon
4)Click on Yes button
5)Click on Allow access button

Steps to deploy web application(.war file):
==========================================
1)open the browser
2)Type the following URL:
  http://localhost:8082/
3)Click on Manager App button
4)Type username:admin
5)Type password:admin
6)Click on Sign in button
7)Scroll down
8)click on choose file button
9)Select the war file(Example:firstservlet.war from firstapp folder)
10)Click on Deploy button.
