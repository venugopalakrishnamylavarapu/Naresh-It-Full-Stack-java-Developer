Servlets API:
============
Servlets API is a Java API that can be used to develop web applications.

1)javax.servlet (or) jakartha.servlet
2)javax.servlet.http (or) jakartha.servlet.http

1)javax.servlet (or) jakartha.servlet package:
==============================================

	Classes				Interfaces
	=======				==========
 1)GenericServlet			1)Servlet
 2)ServletInputStream			2)ServletRequest
 3)ServletOutputStream			3)ServletResponse
 4)ServletException			4)ServletConfig
 5)UnavailableException			5)ServletContext
					6)RequestDispatcher
					7)SingleThreadModel

2)javax.servlet.http (or) jakartha.servlet.http package:
========================================================

	Classes				Interfaces
	=======				===========
1)HttpServlet				1)HttpServlerRequest
2)Cookie				2)HttpServletResponse
					3)HttpSession

jakarta.servlet.Servlet:
=======================
Methods:
1)public abstract void init(ServletConfig) throws ServletException;
2)public abstrac ServletConfig getServletConfig();
3)public abstract void service(ServletRequest,ServletResponse) throws ServletException,IOException;
4)public abstract String getServletInfo();
5)public abstract void destroy();

There are three ways to write servlet program:
1)By implementing jakarta.servlet.Servlet interface.
2)By extending jakarta.servlet.GenericServlet class.
3)By extending jakarta.servlet.http.HttpServlet class.
