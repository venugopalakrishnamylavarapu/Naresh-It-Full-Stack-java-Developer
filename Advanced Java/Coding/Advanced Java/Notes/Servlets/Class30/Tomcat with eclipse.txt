Steps to configure tomcat server in eclipse:
============================================
1)Open JavaEE perspective
2)Click on Servers view
3)Right click in a servers view 
4)Click on New
5)Click on Server
6)Expand Apache
7)Select the Tomcat server which is installed in your system.(in my system-> Tomcat 10.0 version)
8)Click on Next button
9)Click on Browse button
10)Select tomcat installation directory
	(Example:c:\Program Files\Apache Software Foundation\Tomcat 10.0)
11)Click on select folder button
12)Click on next button then  Click on Finish button.

To start tomcat server in eclipse:
=================================
1)Right click on Tomcat version 10.0 server at localhost
2)Click on start

To stop tomcat server in eclipse:
=================================
1)Right click on Tomcat v10.0 Server at localhost 
2)Click on stop

To change the port numbers:
===========================
1)Double click on Tomcat v10.0 server at localhost
2)Move the scroll bar
3)Tomcat admin port:8005 required
4)HTTP/1.1:8082(Here 1024 to 65535 allowed)

Steps to develop web application:
=================================
1)Click on File Menu
2)Click on New
3)Click on Dynamic Web project
4)Type project name(Example:first)
5)Click on Next button
6)Click on Next button
7)Select Generate web.xml deployement descriptor checkbox
8)Click on finish button


Steps to write servlet program:
===============================
1)Right click on first in a project explorer
2)Click on New 
3)Click on Servlet
4)Type Java package(Example:First)
5)Type class name(Example:FirstServlet)
6)Type super class
  (Example:jakarta.servlet.GenericServlet)(for now)
7)Click on Next Button
8)To change the url pattern Click on /FirstServlet in URL mappings box
9)Click on Edit button
10)Type new url patter(Example:/test)
11)Click on Ok button
12)Click on Next button
13)Select the javax.servlet.Servlet interface and click on remove button
14)Click on Finish Button
15)Write on Code editor

To run the above application:
============================
1)Right click on first in a project explorer
2)Click on Run As
3)Click on Run on Server
4)Click on Save button
5)Expand localhost
6)Selct Tomcat v10.0 server at localhost
7)Click on Next button
8)Click on Finish button
9)Type url pattern in address bar(Example:/test)

Steps to change the browser:
===========================
1)Click on Window Menu
2)Click on Web Browser
3)Select the browser u want(Example:Click on Internal Web Browser(Built in eclipse browser))

Second Servlet program:(practice)
======================
import java.io*;
import jakarta.servlet.*;

public class SecondServlet extends GenericServlet
{
	public void service(ServiceRequest req,ServiceResponse res) throws ServletException,IOException
	{
		PrintWriter pw=res.getWriter();
		pw.println("<html><body style=\"display:flex;height:100vh;width:100%;justify-content:center;align-items:center\" bgcolor="blue" textcolor="yellow"><h1>");
		pw.println("Welcome to Servlets");
		pw.println("</h1></body></html>");
	}
}

in the above example,servlet code executed in server & html code executed in browser
